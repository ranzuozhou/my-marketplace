# PR Review Comment Template

> mp-git-review-pr skill 发布 PR 评论时使用此模板。

---

## 模板

```markdown
## 🔍 PR Review — mp-git-review-pr

### 变更概览
- **提交**: {N} 个提交（{commit_types}）
- **文件**: {N} 个文件（+{additions} / -{deletions}）
- **分类**: Plugin {n} | Skill {n} | Config {n} | Docs {n} | Scripts {n} | CI {n} | Other {n}
- **分支状态**: {同步/落后} {base_branch} {N} 个提交

### 评审结果

| # | 检查项 | 结果 | 说明 |
|---|--------|------|------|
| F1 | 分支同步 | ℹ️ | {同步/落后 N 提交} |
| F2 | 变更概览 | ℹ️ | {统计摘要} |
| F3 | Commit 规范 | ✅/❌ | {说明} |
| {D*} | {动态检查项名称} | ✅/⚠️/❌ | {说明} |

### 发现的问题

#### Critical
{阻塞合并的问题，如无则写"无"}

#### Important
{应修复但不阻塞的问题，如无则写"无"}

#### Suggestion
{改进建议，如无则写"无"}

### 建议
{具体行动建议，例如：}
{- 建议运行 `/mp-dev:mp-dev-validate` 校验插件结构}
{- 建议运行 `/mp-git:mp-git-check-merge` 确认合并就绪}

---
*Generated by mp-git-review-pr skill*
```

---

## 文件分类体系

| 分类 | 匹配路径 | 说明 |
|------|---------|------|
| Plugin | `plugin.json` / `.mcp.json` / `CLAUDE.md` / `README.md`（插件级） | 插件元数据和文档 |
| Skill | `skills/**/SKILL.md` + reference files | 技能定义和参考文件 |
| Config | `.claude-plugin/` / `VERSION` / `marketplace.json` | 市场配置文件 |
| Docs | `docs/**` | 项目文档 |
| Scripts | `scripts/**` | 基础设施脚本 |
| CI | `.github/**` | CI/CD 工作流和模板 |
| Other | 不匹配以上任何规则的文件 | 需人工判断的文件 |

---

## 结果图标说明

| 图标 | 含义 | 使用场景 |
|------|------|----------|
| ✅ | 通过 | 检查项完全符合规范 |
| ❌ | 未通过 | 检查项不符合规范（Critical 或 Important） |
| ⚠️ | 需人工判定 | 自动检查无法确定，需要人工确认 |
| ℹ️ | 信息展示 | 仅提供信息，不做合规判定（F1, F2） |
