# Flora PTM Prompt 模板集

本文档包含被三个技能引用的核心 prompt 模板。所有 prompt 均通过 `notebook_query()` 发送给 NLM。

---

## 1. 逐篇分析 Prompt（digest Phase 2）

**用途**: 对 Staging notebook 中的每个 source 进行结构化精华提取。

**默认通用版**:

```
请对这份材料进行深度分析，生成结构化的精简短文：

## 主题与关键词
列出 3-5 个核心关键词

## 核心观点与论证逻辑
概述主要论点及其支撑逻辑链

## 显著亮点
该材料的创新点、独特贡献或最有价值的发现

## 潜在缺陷或局限性
方法论限制、数据局限、未解决的问题

## 一句话核心价值
用一句话概括这篇材料最值得记住的核心价值
```

> 按报告类型的分析变体见 `→ analysis-prompts.md`

---

## 2. 主题提取 Prompt（digest Phase 3）

**用途**: 从所有精简短文中提取关键研究主题，作为 Deep Research 的查询输入。

```
基于所有「精简-*」笔记，提取 5-8 个最重要的研究主题关键词，用逗号分隔，不要解释
```

**Deep Research 查询格式**: `"{extracted_themes} 最新研究进展"`

---

## 3. 关系分析 Prompt（synthesize Phase 1）

**用途**: 发现报告间的主题聚类和关系，输出标准化分组结果。

```
基于所有「精简-*」笔记，分析这些研究报告之间的关系：

## 主题聚类
将报告按主题相似度分组。输出格式：
- 组1: [组名] — 包含报告: [列表]
- 组2: [组名] — 包含报告: [列表]
（如果所有报告属于同一主题，只输出 1 个组）

## 观点对比
哪些报告有互补、矛盾或递进关系？具体说明

## 方法论对比
不同研究采用了哪些不同方法？各有何优劣？

## 信息缺口
这批报告整体覆盖了什么领域？还缺少什么视角？
```

**关键**: 聚类结果中的实际组数 N 直接决定目标 notebook 数量（1-3 个，详见 synthesize Phase 3 分库决策）。

## 4. 综述生成 Prompt（synthesize Phase 2）

**用途**: 生成完整的跨文献综述，综合所有报告的核心洞察。

```
基于所有 source 和分析笔记，生成完整的跨文献综述：

## 研究领域全景概述
这批材料整体覆盖了什么领域和主题？

## 主要研究流派与观点分类
按流派或立场分类梳理

## 共识与争议点
哪些结论有共识？哪些存在分歧？

## 发展趋势
从时间线看，这个领域的发展方向是什么？

## 综合结论
综合所有材料后的整体结论和洞察
```

## 5. 导航笔记模板（synthesize Phase 4）

**用途**: 为每个目标 notebook 生成内容索引和推荐阅读顺序。

```markdown
# 导航 — {topic}/{subtopic}

## Source 来源映射
| 序号 | Source 名称 | 原始来源 | 类型 |
|------|------------|---------|------|
| 1 | {source_name} | {original_path_or_url} | {原文/精简/综述/关系} |

## 推荐阅读顺序
1. 先读 `00b-综述-{topic}` — 建立全局视角
2. 再读 `00c-关系-{topic}` — 理解报告间关系
3. 按兴趣选读 `00a-精简-*` — 了解各篇要点
4. 最后深读感兴趣的原文 source

## 主题概要
{subtopic 的一句话描述}
```

## 6. 社区高级分析模式（可选）

以下 7 种分析模式来自社区最佳实践，高级用户可用来替代默认 prompt：

### 6.1 交叉综合

```
将所有报告的核心论点进行交叉对比，识别：
1. 共同引用的理论基础或数据源
2. 相互支撑或矛盾的实验结果
3. 可以组合形成新洞察的跨报告发现
```

### 6.2 矛盾检测

```
逐对比较报告的核心结论，识别：
1. 直接矛盾的观点及其各自论据强度
2. 看似矛盾但实际互补的发现（不同适用条件）
3. 未被任何报告讨论但逻辑上应存在的矛盾
```

### 6.3 研究空白

```
基于所有报告的覆盖范围，识别：
1. 所有报告均未涉及但在该领域重要的子话题
2. 被多篇提及但无深入研究的方向
3. 现有研究方法论的共同盲区
```

### 6.4 趋势分析

```
按时间线梳理所有报告，分析：
1. 核心概念/方法的演进路径
2. 研究重心的迁移方向
3. 新兴话题与衰退话题的信号
```

### 6.5 方法论对比

```
提取每篇报告使用的研究方法，对比：
1. 定性 vs 定量方法的分布
2. 不同方法得出的结论差异
3. 各方法的适用场景和局限性
```

### 6.6 反向提问

```
从批判性角度审视所有报告：
1. 每篇报告未回答的关键问题是什么？
2. 哪些假设未经验证就被接受？
3. 如果要反驳这些报告的核心结论，最有力的论据是什么？
```

### 6.7 遗漏洞察

```
假设你是该领域的资深研究员，审视这批报告后：
1. 这些报告作为整体遗漏了什么关键视角？
2. 有哪些跨学科的关联被忽视？
3. 从实践角度看，这些研究结果有哪些未被讨论的应用场景？
```

**使用方式**: 在 digest Phase 2 人工判断点，用户可选择用社区模式替代默认分析 prompt 对特定报告进行分析。在 synthesize Phase 1，可用社区模式补充关系分析。
